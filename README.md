*Домашнее задание курса SkyPro, модуля 14.1 (Введение в ООП).

## Установка:

1. **Клонируем репозиторий:**
```
git clone https://github.com/soulbratan/SkripnikovOV_OOP.git
```
2. **Устанавливаем зависимости:**
```
poetry install
```

## Примеры использования:
1. Созданы классы для продуктов (атрибуты: название, описание, цена, количество) и категорий(атрибуты: название, описание, список продуктов в категории).
2. В модуле 'utils.py' определены функции 'read_json' - чтение данных из json-файла, и 'create_objects_from_json' - создание объектов класса из данных json-файла.
3. 20.05.25 - В описании класса 'Category' реализован метод 'add_product' для добавления продукта (в качестве аргумента) в приватный атрибут products.
4. 20.05.25 - В описании класса 'Category' реализован геттер 'products', возвращающий список товаров в виде строк.
5. 20.05.25 - В описании класса 'Product' создан класс-метод 'new_product', который принимает на вход параметры товара в словаре и возвращает созданный объект класса Product.
6. 20.05.25 - В описании класса 'Product' добавлен сеттер для 'price' и реализована проверка (в случае если цена равна или ниже нуля, выводите сообщение в консоль 'Цена не должна быть нулевая или отрицательная')
